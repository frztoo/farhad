## Upload to InfinityFree
1) Download the ZIP.
2) In InfinityFree File Manager (or FTP), open:
   `htdocs/`
3) Upload **all** files and folders from the ZIP into `htdocs/`
4) Make sure `index.html` is inside `htdocs/`
5) Visit your domain: https://farhadgq.infinityfreeapp.com

## Customize
- Replace placeholder shop links in: `index.html` + category pages
- Update contact info in: `contact.html`
- Replace imprint/privacy templates with your legal text

## Contact form
This demo form is front-end only.
To make it send emails quickly, use Formspree:
- Create a free endpoint
- Paste the endpoint URL into the form's `action=""` in contact.html
